### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/pawel-schmidt/the-right-choice.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Open the project
   ```sh
   npm run dev
   ```
